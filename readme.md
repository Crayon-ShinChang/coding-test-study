# 코딩 테스트 테스트

## Conventions
### Branch NAme
- `{주차}/{본인_깃허브_아이디}` 형식으로 네이밍합니다.
- ex) 1주차의 경우 `w1/initmumu`

### Commit Message
- 커밋 메시지는 `{문제번호}/{문제명}/{ms(실행시간)}` 형식으로 작성합니다.
- commit type은 생략합니다.
- ex) `1342/폴리오미노/148ms`

### Folder Structure
- 대분류: 주차/중분류: 문제번호/ 파일명: 본인 깃허브 아이디
- ex) `w1/1001/foobar.cpp`
